# GUI Basic steps

Here, we describe basic steps to pay attention to while using HRaDeXGUI.
For more detailed description, check the helpers in the application
(indicated by circled quesition marks next to elements) or read the
documentation e.q.
[`vignette("gui_guide")`](https://hadexversum.github.io/HRaDeX/articles/gui_guide.md).

## Upload file

Upload datafile and check the message below.

![](figures/hradex_upload_data.png)

## Check the selected state

HrRaDeX run single-state analysis, so make sure the correct state is
chosen.

![](figures/hradex_check_state.png)

## Check the controls

As the recommended data format is fractional, make sure that the
undeuterated and fully deuterated control are selected correctly.

![](figures/hradex_check_controls.png)

## Check exchange groups

Next, scroll down the settings panel and check if the exchange group
limits are correct. You may change them, if the analysis results are
unsatisfatory.

![](figures/hradex_check_groups.png)

## Run the analysis

If all the settings are acceptable, run the analysis by clicking the
button.

![](figures/hradex_run.png)

Enjoy the results!
