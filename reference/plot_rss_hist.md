# Histogram of fit rss values

Histogram of rss values for selected models for each peptide. Divided
with respect to the number of model parameters. Plot to judge the
goodness of the models.

## Usage

``` r
plot_rss_hist(fit_values, interactive = FALSE)
```

## Arguments

- fit_values:

  data.frame with fit values

- interactive:

  ...

## Value

a ggplot2 object. Needs converstion to girafe for the interactive mode.
