# Plots the uc difference type 1

Function plots the uc distance from uc_distance_dataset. Be careful, as
uc_distance_dataset contains distances calculated using different
methods, as described in vignette TODO.

## Usage

``` r
plot_uc_real_dist(
  uc_distance_dataset,
  squared = FALSE,
  fractional = TRUE,
  interactive = FALSE
)
```

## Arguments

- uc_distance_dataset:

  ...

- squared:

  indicator if the uc distance is squared

- fractional:

  ...

- interactive:

  logical, true for GUI

## Value

a ggiraph object.

## See also

create_uc_distance_dataset
