# Prepare coloring commands for ChimeraX

This function prepares the coloring commands for ChimeraX. If you wish
only to create coloring commands and downlaod them as a text file, see
`export_chimera_commands` function.

## Usage

``` r
prepare_chimera_commands(
  hires_params,
  dominant = FALSE,
  state = hires_params[["State"]][1],
  chains = c("A")
)
```

## Arguments

- state:

  biological state

- chain:

  chain for which the coloring commands are created
