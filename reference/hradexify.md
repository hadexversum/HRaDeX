# HRaDeX customized ggplot theme

This function HRaDeXifies plot. It adds HRaDeX logo and ggplot theme.

## Usage

``` r
hradexify(plt)
```

## Arguments

- plt:

  ggplot object. Plot to HRaDeXify.

## Value

a `ggplot2` object.

## Details

Function adds the logo of HRaDeX package in the left down corner of the
plot and the HRaDeX theme. This function is adapted from the HaDeX
package.
